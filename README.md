## PROJECT OVERVIEW

📌 **프로젝트 이름**

펀드림
<br>
(FunDream)

<br><br>

📌 **프로젝트 목적**

사용자가 관심 있는 상품이나 아이디어에 **직접 펀딩**하여,  
판매자가 **목표 금액을 달성하면 제작 및 배송**이 이루어지는  
**크라우드 펀딩 플랫폼**을 개발합니다.  
<br><br>
